# Apunts de Llenguatges de Marques (DAW)

Benvingudes i benvinguts als apunts del mòdul **Llenguatges de marques i sistemes de gestió d’informació** del cicle de **Desenvolupament d’Aplicacions Web (DAW)**.

!!! info "Què trobaràs en aquesta web?"
    - Una introducció al mòdul de Llenguatges de Marques.

???+ tip "Com navegar pels apunts"
    Utilitza el **menú de navegació** superior i lateral per moure’t entre els diferents temes.  
    També pots buscar conceptes concrets amb el quadre de **búsqueda** que apareix a la part superior de la pàgina.

Per a canviar entre **tema clar** i **tema fosc**, fes servir el botó amb la icona de sol/lluna a la barra superior.
